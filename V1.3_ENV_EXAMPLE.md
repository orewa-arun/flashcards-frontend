# v1.3 Environment Variables Configuration

Add these variables to your `backend/.env` file:

## PostgreSQL Database

```bash
POSTGRES_URL=postgresql+asyncpg://postgres:postgres@localhost:5432/self_learning_ai
```

## Cloudflare R2 Configuration

```bash
R2_ACCOUNT_ID=your_account_id
R2_ACCESS_KEY_ID=your_access_key_id
R2_SECRET_ACCESS_KEY=your_secret_access_key
R2_BUCKET_NAME=course-content
R2_ENDPOINT_URL=https://your_account_id.r2.cloudflarestorage.com
```

## AI Model Configuration

These are optional - defaults are already set in `backend/app/config.py`:

```bash
# Structured Analysis Model (default: gemini-2.0-pro-exp)
MODEL_ANALYSIS=gemini-2.0-pro-exp

# Flashcard Generation Model (default: claude-3-haiku-20240307)
MODEL_FLASHCARDS=claude-3-haiku-20240307

# Quiz Generation Model (default: gemini-2.0-flash-exp)
MODEL_QUIZ=gemini-2.0-flash-exp
```

## Complete .env Template

```bash
# PostgreSQL Database
POSTGRES_URL=postgresql+asyncpg://postgres:postgres@localhost:5432/self_learning_ai

# MongoDB (existing)
MONGODB_URL=mongodb://localhost:27017
DATABASE_NAME=study_analytics

# Cloudflare R2 Configuration
R2_ACCOUNT_ID=your_account_id
R2_ACCESS_KEY_ID=your_access_key_id
R2_SECRET_ACCESS_KEY=your_secret_access_key
R2_BUCKET_NAME=course-content
R2_ENDPOINT_URL=https://your_account_id.r2.cloudflarestorage.com

# AI Model Configuration (Override defaults)
MODEL_ANALYSIS=gemini-2.0-pro-exp
MODEL_FLASHCARDS=claude-3-haiku-20240307
MODEL_QUIZ=gemini-2.0-flash-exp

# API Configuration (existing)
API_HOST=0.0.0.0
API_PORT=8000
DEBUG=True

# CORS Configuration (existing)
ALLOWED_ORIGINS=http://localhost:5173,http://localhost:3000

# RAG Backend Configuration (existing)
RAG_API_BASE_URL=http://localhost:8001
```

## Notes

- Replace `your_account_id`, `your_access_key_id`, and `your_secret_access_key` with actual Cloudflare R2 credentials
- The PostgreSQL database will be automatically created with tables on first startup
- Model names can be changed to use different AI models as they become available
